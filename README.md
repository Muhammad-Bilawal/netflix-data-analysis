# Netflix Data Analysis using Python

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on a Netflix dataset using Python.
The goal is to understand the distribution of movies and TV shows, top countries producing content, and popular genres.

---

## 📊 Dataset

The dataset contains information about Netflix titles, including:

* Type (Movie or TV Show)
* Title
* Country
* Genre
* Release Year

---

## ⚙️ Tools & Technologies

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

---

## 🔍 Analysis Performed

* Data loading and inspection
* Checking missing values
* Analysis of content types (Movies vs TV Shows)
* Top countries producing content
* Most common genres

---

## 📈 Visualizations

* Bar chart of Movies vs TV Shows
* Bar chart of Top 5 Countries

---

## 📌 Key Insights

* Movies are more common than TV shows on Netflix
* The United States produces the highest number of titles
* Netflix content comes from multiple countries

---

## ▶️ How to Run

1. Download the dataset (netflix1.csv)
2. Open the notebook (netflix_analysis_final.ipynb)
3. Run all cells in Jupyter Notebook
